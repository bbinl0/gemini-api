Gemini API

Start Chatting

Available Models:

  • Gemini 2.5 Pro (Standard)
    - সর্বাধিক শক্তিশালী এবং বহুমুখী মডেল। জটিল যুক্তি, কোডিং এবং মাল্টিমোডাল কাজের জন্য উপযুক্ত।
  • Gemini 2.5 Flash (Fastest)
    - দ্রুত এবং সাশ্রয়ী মডেল। দ্রুত প্রতিক্রিয়া প্রয়োজন এমন কাজের জন্য আদর্শ।
  • Gemini 2.5 Flash Lite (Fastest)
    - সবচেয়ে হালকা এবং সম্ভবত দ্রুততম মডেল। অত্যন্ত দ্রুত প্রতিক্রিয়া প্রয়োজন এমন ক্ষুদ্র কাজের জন্য।
  • Gemini Live 2.5 Flash Preview (Fast)
    - লাইভ ইন্টারেকশন বা রিয়েল-টাইম ব্যবহারের জন্য দ্রুত মডেলের প্রাকদর্শন।
  • Gemini 2.5 Flash Native Audio (Fast)
    - অডিও ভিত্তিক ডায়ালগ বা কথোপকথনের জন্য বিশেষায়িত দ্রুত মডেল।
  • Gemini 2.5 Flash Audio (Experimental) (Fast)
    - অডিও এবং থিঙ্কিং ডায়ালগের জন্য একটি পরীক্ষামূলক মডেল।
  • Gemini 2.5 Flash Image Preview (Fast)
    - ইমেজ ভিত্তিক কাজের জন্য দ্রুত মডেলের প্রাকদর্শন।
  • Gemini 2.5 Flash TTS (Fast)
    - টেক্সট-টু-স্পিচ (TTS) প্রিভিউয়ের জন্য দ্রুত মডেল।
  • Gemini 2.5 Pro TTS (Standard)
    - টেক্সট-টু-স্পিচ (TTS) প্রিভিউয়ের জন্য শক্তিশালী মডেল।
  • Gemini 2.0 Flash (Fast)
    - দ্রুত এবং সাশ্রয়ী মডেলের পূর্ববর্তী সংস্করণ।
  • Gemini 2.0 Flash Image Gen (Fast)
    - ইমেজ তৈরির জন্য দ্রুত মডেলের প্রাকদর্শন।
  • Gemini 2.0 Flash Lite (Fastest)
    - দ্রুততম মডেলের পূর্ববর্তী সংস্করণ।
  • Gemini 2.0 Flash Live (Fast)
    - লাইভ ইন্টারেকশনের জন্য দ্রুত মডেলের পূর্ববর্তী সংস্করণ।
  • Gemini 1.5 Flash (Fastest)
    - বর্তমানে উপলব্ধ সবচেয়ে দ্রুত এবং দক্ষ মডেল।
  • Gemini 1.5 Flash 8B (Fast)
    - আলাদা প্যারামিটার সহ একটি হালকা মডেল।
  • Gemini 1.5 Pro (Standard)
    - বর্তমানে উপলব্ধ সবচেয়ে শক্তিশালী এবং বহুমুখী মডেল।

Example Usage:

To use the gemini-2.5-pro model with the prompt "Hello world", send a POST request to the following URL:

https://gemini-api-silk-two.vercel.app/generate/gemini-2.5-pro

The request body should be a JSON object with the following structure:

{
    "prompt": "Hello world",
    "history": []
}
